
# Chrome Extensions  **SwitchyOmega** 设定备份文件
# Proxy Addon -  Omega online backup
#### 介绍
适用于类似SwitchyOmega的插件，如：SwitchyOmega，Proxy SwitchyOmega 3等。可以在Chrome、MS Edge和夸克浏览器上使用。

#### 备份和恢复的方法
打开插件的选项卡，定位到“设定” -> “导入/导出”


#### 包含的设定

1.  Moravia Proxy
2.  SDL Proxy
3.  V2Ray Proxy
4.  WARP+ Proxy
5.  S2V
6.  auto switch
